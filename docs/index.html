Pretty self-explanatory.

List some genes of interest

Enter a list of genes (Ensembl or Entrez IDs or symbols, one per line, text after # is ignored) and we'll find any gene sets with overlaps. Sets will be ranked by the enrichment p-value.

Search gene set names/descriptions

Restrict the search to sets with names or descriptions containing all search terms. * and ? wildcards are supported.

Restrict gene set collections

Limit the search to particular gene set collections.
