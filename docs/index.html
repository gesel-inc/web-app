Bioconductor gene set search

Search on genes of interest

Search for sets that overlap any of the specified genes.

  • Ensembl or Entrez IDs or gene symbols are supported.
  • One gene should be provided per line, any text after # will be ignored.
  • Sets will be ranked by the enrichment p-value in the output table.

Search gene set names/descriptions

Restrict the search to sets with names or descriptions containing all search terms. * and ? wildcards are supported.

Limit the search to particular gene set collections.
